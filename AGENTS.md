# Project Instructions

- After making changes, always finish with `git add -A`, `git commit -m "<message>"`, and `git push` to `main`.
- Preferred shortcut: run `npm run codex:auto-push` to execute `git add -A && git commit -m "auto: codex changes" && git push`.
- Codex should treat the final git commit and push as part of every session's completion workflow for this repository.
